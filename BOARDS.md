# The Gate · honest self-checks + the one real independence

Adapts Kailin's Ship Gate standard (3 reviews, no slop) to content. Be honest about what is and is not independent.

## What these checks actually are
The three boards below are **SELF-checks**: one model (the running session) applying three lenses. They catch a lot, but a model re-grading its own work with the first scores still in context is NOT independent. Do not call it that.

**The only real independent checks:**
1. **Fresh sub-agent re-score** (Agent tool) for **Plated-feed-bound clips only** (keeps the daily run light, this is the tier where a miss is most costly and public). The sub-agent gets ONLY the frames/transcript and the rubric, no prior scores. If its score disagrees widely with the self-check, flag the clip for Kailin instead of auto-bucketing. Personal-feed and below rely on the self-checks plus the lite cadence; a full re-score everywhere is the opt-in "deep run".
2. **Kailin's eyes** are the named final gate on anything going to the **Plated feed**. The skill does not self-certify Plated-feed taste.

## The tiered floor (tuned for a pre-launch sprint)
Pre-launch, cadence beats pristine: nobody converts on a feed that never posts. So the Plated-feed floor is set to ship real-and-good, not unicorn-perfect.

| Destination | Floor | Checks run |
|---|---|---|
| **Plated Circuit feed** | **90+** (was 95; lowered for the sprint so the event feed is not starved) | All 3 self-checks + sub-agent re-score + Kailin's eyes |
| **Personal IG feed** | ~85, authenticity premium applies | All 3 self-checks + sub-agent re-score |
| **TikTok / Reels** | ~75, raw-native | Board 1 + Board 3 |
| **IG / personal Story** | ~65, volume | Single pass |

Below floor → route to Polish with the objection, OR (non-event only) drop a tier and re-judge. Event-critical never drops, it goes to Polish.

## Board 1 · Platform-Native Lens
Would a real creator post this. Hook in 1 to 2 seconds, scroll-stop, retention, native feel.
- **Can FAIL a clip for being TOO polished / ad-like on personal or TikTok.** Over-produced personal content is a miss, not a win.

## Board 2 · Brand & Strategy Lens
- Footage: checks VIBE and context fit for the destination page (NOT hex codes, raw footage has no palette).
- Designed overlays / covers / Nanobanana station art ONLY: run `brand-guard` + `brand-lint.py <biz> <file>` for hex/font drift (Plated brass #B8966A / cream #F4EFE6, not District burgundy).
- Personal-first check: "Plated-specific, or defaulting to Plated out of habit?" If habit → personal.
- Serves the goal: for Plated, does it move July 30 forward.

## Board 3 · Steve Jobs Final
Taste gate. Good, or just not bad. Enforces real-content-is-king, kills slop and over-AI'd clips, protects the personal brand from looking generic. Final say at its tier.

## Pass / fail
- Meets floor + passes required checks → ready, on the sheet.
- Fails → Polish with the specific objection, or (non-event) drop a tier.
- Cannot clear even the Story floor → Hold/Kill with the reason written (Orphan Audit, nothing silent).
