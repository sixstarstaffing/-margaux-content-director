# Routing Tree · personal-first, conversion-aware, series-aware

Default bias: when a clip fits both pages, **personal wins.** Plated earns a clip with clear event/brand fit. Personal page is the high-volume engine, Plated stays curated and event-pointed.

Run per asset after scoring and hook engineering.

```
START
 │
 ├─ Is it Plated-specific? (event, food, Plated world, on Plated brand)
 │     YES → candidate for PLATED. Board 2 confirms ("really Plated-specific,
 │           or defaulting out of habit?"). If habit → PERSONAL.
 │     NO  → PERSONAL (personal-first default).
 │
 ├─ Platform by scores (+ authenticity premium):
 │   High Tech (70+) + High Content (70+)      → FEED on the chosen page.
 │   Low Tech (<70)  + High Content (70+)       → TIKTOK / REELS (raw is native).
 │   High Tech       + Low Content (<70)        → STORY, or hold as B-ROLL.
 │   Fixable craft issue (trim/stabilize/caption/color) → POLISH QUEUE (write the fix).
 │   Low Tech + Low Content                     → HOLD / KILL (write the reason).
 │
 ├─ EVENT-CRITICAL OVERRIDE:
 │   If the clip is genuinely Plated/event content but below the feed bar,
 │   route to POLISH and KEEP it on Plated. Do NOT auto-drop event content
 │   to TikTok/Story. Only NON-event content drops tiers freely for cadence.
 │
 ├─ Tag the SERIES (FORMATS.md): which recurring format + episode.
 ├─ Assign the CONVERSION ROLE (FORMATS.md funnel): awareness / build / soft-CTA / hard-CTA.
 │
 └─ Exit with: destination + platform + series + conversion role + reason. No exceptions (Orphan Audit).
```

## Tie-breakers
- Fits both pages equally → personal.
- Event-urgent AND personal → Plated gets it for the 30-day window, note it.
- Borderline feed vs TikTok on NON-event content → drop a tier, cadence beats perfection on raw platforms.
- Great clip, wrong aspect/format → route to the platform whose format it already fits.

## Cadence vs the event goal
Non-event clips that miss the feed bar drop a tier (feed → TikTok → Story) same day to keep the personal page fed. Event-critical clips never drop out of Plated, they get polished and kept. Cadence protects volume, the override protects the launch.
