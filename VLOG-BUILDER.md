# VLOG BUILDER (Margaux module)
**The repeatable process for turning a voiceover/script + footage into a finished narrated vlog. Proven on the Plated tickets vlog (2026-07-01). Use for any BrandForge client, adapt to their brand + voice.**
*This is a REAL skill, not improvisation. Follow the steps in order. It pairs the content-director (footage) + copywriter (words) skills.*

## WHEN TO USE
Client wants a narrated video: a tickets/offer vlog, a founder story, a "how it's made," a passport/product reveal. Anytime there's a VOICEOVER or script + a pile of footage. (For a raw talking-head where they film themselves talking, that's Captions.ai auto-caption territory, not this.)

## THE 8-STEP PROCESS
### 1. SCRIPT / VOICEOVER FIRST (the spine)
The voiceover is the backbone. Either the client records it word-for-word (they rehearse, not improvise), or write it in their voice via the `copywriter` skill (VOC, no em-dashes, one CTA, keyword-rich for SEO). Keep it real and casual. Trim dead air + Studio Sound it (Descript).

### 2. B-ROLL MAP (the core move)
Break the VO into BEATS, one per sentence/idea. For EACH beat, name the exact clip that shows what they're saying. Deliver a timecode table: `beat / ~time / the line / the b-roll clip / on-screen note`. This is what makes it feel intentional, not stock.
- REAL footage = the hero, use it as-is (it already moves, has its own audio). Never run real footage through AI.
- Flag any beat with NO matching footage as a gap + suggest the nearest cover (a related still, a brand graphic), never leave a line on a black frame.

### 3. LUMA ANCHORS (stills -> motion)
Any STILL that needs to move (offer slides, tier cards, country/product stills) gets animated in Luma (image-to-video, motion strength LOW so text stays crisp). These become the GRAPHIC ANCHORS: open on one, close on one, punctuate the middle. Luma is ONLY for stills, never for real footage.

### 4. ONE DRIVE FOLDER, NUMBERED (no hunting)
Gather every clip the map needs, real b-roll + Luma anchors, into ONE Drive folder, renamed in timeline order (01-, 02-, ...). The client grabs the whole folder and uploads in order. This is the "don't make me hunt" rule, do the sorting for them.

### 5. ASSEMBLE
- **Descript** for VO + b-roll vlogs (the default here): import the VO -> it transcribes -> that's the timeline -> Studio Sound -> drop b-roll over each beat (silent clips + stills are FINE in Descript) -> auto-subtitles -> export 1080x1920.
- **Captions.ai** only for talking-head (client on camera). Note: Captions rejects silent clips, Descript does not, another reason Descript wins for VO+b-roll.
- Video holds 3-4s, photos flash ~1.5s with a slow push-in (never static). Hard cut on the beat.

### 6. CAPTION + DISCOVERY (the SEO layer)
- Feed caption via `copywriter` skill: their voice, one CTA, no em-dashes, keyword-rich.
- Turn ON full VO subtitles (not just scene labels), Instagram reads them.
- Alt text + on-screen text + caption + the spoken VO should all name the SAME keywords = reinforcement = discovery.
- Music: the voiceover carries it. If music-only, use a trending sound at 5,000-30,000 uses (sweet spot), posted in its first 24-48 hrs.

### 7. REVIEW GATE (95+, no mercy)
Before ship, score it: (a) does every b-roll clip match its VO line, (b) hook in the first 1-2 seconds, (c) tight enough to hold watch-time (the #1 IG signal), (d) on-brand (palette/voice/fonts), (e) firewall-safe (no forbidden terms), (f) zero em-dashes in any burned text, (g) one clear CTA. Fix anything under 95 or re-cut.

### 8. ROUTE (personal-first)
Feed vs story vs highlight. Evergreen offers (tickets, tiers) = a HIGHLIGHT that lives on the profile + a feed post. Time-sensitive pushes = feed + story. Tie goes to the personal/founder brand.

## TOOL RULES (memorize)
- **Descript** = voiceover + b-roll vlogs. **Captions.ai** = talking-head. **Luma** = stills -> motion only. Real footage = never AI'd.

## REFERENCE IMPLEMENTATION
The Plated tickets vlog (2026-07-01): 45s VO -> b-roll map (`_Plated/TICKETS-VLOG-BROLL-MAP.txt`) -> Luma-animated ticket slides as anchors + real passport/rooftop b-roll -> one Drive folder -> Descript assemble -> caption via copywriter -> shipped. Passport vlog uses the same process.

## PER-CLIENT
This process is client-agnostic. Swap in each client's brand kit, voice source, and footage. Living skill, refine it as we learn what converts per client + industry (same standing directive as the copywriter skill).
